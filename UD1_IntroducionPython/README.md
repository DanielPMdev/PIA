# Introducción a Python y Librerías Base

> Asignatura: Programación de Inteligencia Artificial  
> Curso de Especialización en Inteligencia Artificial y Big Data

## 📌 Descripción
Este primer bloque temático sienta las bases de programación necesarias para el desarrollo de Inteligencia Artificial. Cubre desde conceptos de Programación Orientada a Objetos (POO) en Python, como la herencia y los decoradores, hasta el dominio de las principales librerías de análisis y visualización de datos que se usarán en todo el curso.

## 🎯 Objetivos de Aprendizaje
- Comprender y aplicar programación orientada a objetos avanzada, incluyendo herencias y decoradores.
- Manipular grandes volúmenes de datos mediante arrays multidimensionales y estructuras tabulares (DataFrames y Series).
- Implementar técnicas de visualización de datos efectivas usando histogramas y diagramas de dispersión.
- Resolver problemas de lógica y estructuración de datos empleando librerías especializadas.

## 🧠 Contenidos

### Fundamentos de Python
A lo largo de estos ejercicios, se exploran métodos, atributos, la palabra clave `self`, decoradores (syntactic sugar) y la herencia de clases (reutilización y sobrescritura de métodos). Se incluye también un caso práctico integrador: desarrollo de una Agenda Digital.

### Análisis y Manipulación de Datos
- **NumPy**: Creación de arrays uni y multidimensionales, indexación, *broadcasting* y operaciones estadísticas fundamentales.
- **Pandas**: Dominio de objetos `Series` y `DataFrame`. Acceso, modificación, cruce, evaluación de expresiones complejas y gestión de archivos.

### Visualización y Casos Prácticos
- **Matplotlib**: Representación gráfica de datos, incluyendo scatter plots y creación de gráficas a partir de DataFrames.
- **Evaluaciones**: Retos y prácticas puntuables integradoras sobre casos reales (e.g., dataset del Titanic) para consolidar los conocimientos.

## 🛠️ Tecnologías y Librerías
